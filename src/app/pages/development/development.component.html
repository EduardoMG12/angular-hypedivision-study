Tema visual consistente

O design neon/cyberpunk exigiu uma abordagem cuidadosa para manter uma experiência visual coesa. Resolvi isso criando um sistema de design baseado em variáveis CSS e utilitários Tailwind personalizados.

Animações performáticas

Implementei animações para os flashcards que funcionam bem mesmo em dispositivos menos potentes, utilizando transforms CSS e gerenciando cuidadosamente as renderizações.

Gerenciamento de estado

Para gerenciar o estado dos flashcards e decks, implementei React Query para operações de dados e Context API para estado global, resultando em um código mais limpo e manutenível.
